Tauron G13 buy tariff generator — Reads the Tauron G13 buy tariff setup and generates quarter-hour buy-price files for automation.

What this script does

The script reads the Tauron G13 input table from the setup file, expands it into quarter-hour slots, applies holiday/weekend flags and writes the current buy-energy tariff table.

Status and HTML handling

This script does not generate the report HTML manually. At the end of the run it calls the local NIA script-status builder from /work/00_nia_system/00_10_nia_system_statuses/00_10_10_status_builder_local/. The global builder later creates registry and script_info pages from the generated status metadata and these INI sections.

Main outputs

Main CSV: 05_energy_buy.csv in the parent tariff directory. Central automation data is updated only in owned columns: buy_energy and buy_distribution.
