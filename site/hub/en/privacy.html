Website information

Privacy

The website is designed to provide information without unnecessary collection of user data.

On this page:Processing scopeCookies and analyticsContact
01
Processing scope

An information website without user accounts

The site has no registration, user dashboard or contact form.

When using the site, users are not asked to provide a name, email address or other personal data. Contact takes place outside the website using the email address provided.

The server operator may process standard technical log data such as an IP address, connection date and time, requested resource and browser information. The scope and retention period depend on the configuration of the final hosting environment.

02
Cookies and analytics

No marketing tracking

The site does not use web analytics, advertising, user profiling or third-party social media widgets.

The current version does not set first-party cookies or store data in the browser’s local storage. It therefore does not display a consent banner.
